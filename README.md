# Tabata Timer

A mobile-first interval timer for Tabata workouts.

## Features

- Customizable work/rest intervals
- Configurable rounds and sets
- Audio cues on phase transitions
- Dark mode UI with orange accent
- Mobile-optimized (portrait only)

## Usage

Open `index.html` in a browser.

### Controls

- **Start/Pause** - Begin or pause the timer
- **Reset** - Return to initial state
- **Settings (sliders icon)** - Customize work time, rest time, rounds, and sets

### Default Settings

| Setting | Default |
|---------|---------|
| Work | 20 sec |
| Rest | 10 sec |
| Rounds | 8 |
| Sets | 1 |

## Tech

- Single HTML file
- Tailwind CSS v4 (CDN)
- Web Audio API (beeps)
- No build step required
